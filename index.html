Fundamentos de Machine Learning

O que é Inteligência Artificial?

Inteligência Artificial (IA) busca criar sistemas capazes de realizar tarefas que normalmente exigiriam inteligência humana.

Isso inclui:

  • reconhecimento de padrões
  • tomada de decisão
  • previsão de resultados

Existem diferentes formas de construir esses sistemas.

Uma abordagem tradicional é definir regras manualmente.

Machine Learning segue um caminho diferente:

em vez de programar regras, o sistema aprende essas regras a partir de dados.

---

Como o Machine Learning funciona?

O aprendizado de máquina consiste em aprender uma relação entre entradas e saídas.

\( x \rightarrow y \)

  • \(x\): dados de entrada (features)
  • \(y\): saída desejada (target)

O objetivo é encontrar uma função que produza boas previsões:

\( \hat{y} = f(x) \)

---

Indução de hipótese

Esse processo pode ser entendido como indução de hipóteses.

Ou seja:

  • observamos exemplos
  • inferimos uma regra geral

Chamamos essa regra de hipótese:

\( h(x) \)

O modelo escolhe, entre várias possíveis funções, aquela que melhor explica os dados.

---

Espaço de hipóteses

Não buscamos qualquer função possível, mas sim dentro de um conjunto limitado.

Esse conjunto é chamado de espaço de hipóteses.

  • funções lineares
  • árvores de decisão
  • redes neurais

Escolher o modelo significa escolher esse espaço.

---

Como escolher a melhor hipótese?

Precisamos de um critério para avaliar quão boa uma hipótese é.

Isso é feito medindo o erro entre:

  • valor real
  • valor previsto

O modelo ajusta seus parâmetros para minimizar esse erro.

---

Generalização

O objetivo não é apenas ir bem nos dados observados.

Queremos que o modelo funcione bem em dados novos.

Isso é chamado de generalização.

---

Resumo do processo

  • definir um espaço de hipóteses
  • usar dados para escolher a melhor hipótese
  • garantir que ela generalize para novos dados

Esse processo conecta todos os conceitos que serão estudados a seguir.
