Gradient Boosting

1º - O Problema: Prever Nota

Considere o mesmo cenário de regressão:

Horas (\(x\)) Nota (\(y\))
12
23
35
44
56

Nosso objetivo é prever a nota com base nas horas de estudo.

2º - Ideia do Gradient Boosting

Em vez de criar várias árvores independentes (como no Random Forest), o Gradient Boosting constrói modelos sequencialmente.

  • Cada novo modelo corrige os erros do anterior
  • O foco está nos resíduos (erros)

3º - Modelo Inicial

Começamos com uma previsão simples: a média dos valores.

\( \bar{y} = 4 \)

Primeira previsão para todos os pontos:

\( \hat{y}_i = 4 \)

4º - Cálculo dos Erros (Resíduos)

\(x\) \(y\) \(\hat{y}\) Erro (\(y - \hat{y}\))
124-2
234-1
3541
4440
5642

5º - Treinando uma Árvore nos Erros

Agora treinamos uma árvore para prever os erros (resíduos). 

        [Horas ≥ 3]
          /      \
       Não        Sim
      /            \
   -1.5          +1.0

Esses valores representam a média dos resíduos em cada grupo.
Ou seja, a árvore aprende qual é o erro médio que precisa ser corrigido em cada região.

6º - Atualizando as Previsões

Nova previsão:

\( \hat{y}_{novo} = \hat{y}_{antigo} + \alpha \cdot \text{árvore} \)

O parâmetro \(\alpha\) é chamado de Learning Rate (ou Shrinkage).

  • Controla o quanto cada árvore influencia o modelo final
  • Valores menores tornam o aprendizado mais gradual
  • Ajuda a evitar overfitting

Considerando \(\alpha = 0.5\):

  • Para \(x < 3\): \(4 + 0.5(-1.5) = 3.25\)
  • Para \(x ≥ 3\): \(4 + 0.5(1.0) = 4.5\)

7º - Repetindo o Processo

Calculamos novos resíduos e treinamos outra árvore.

A cada iteração, os resíduos tendem a ficar mais próximos de zero,
indicando que o modelo está melhorando suas previsões. 

Modelo 1 → média
Modelo 2 → corrige erros
Modelo 3 → corrige erros restantes
...

8º - Intuição Final

  • Os modelos são construídos em sequência
  • Cada um aprende com os erros anteriores
  • O resultado final é a soma de vários modelos fracos

9º - Diferença para Random Forest

  • Random Forest: árvores independentes (paralelo)
  • Gradient Boosting: árvores sequenciais (dependentes)

10º - Conclusão

Gradient Boosting transforma modelos simples em um modelo forte ao focar iterativamente nos erros.

Essa ideia é a base de algoritmos modernos como XGBoost e LightGBM.
