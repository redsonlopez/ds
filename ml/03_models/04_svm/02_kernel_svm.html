Support Vector Machine (SVM) - Kernel

1º - O Problema: Dados Não Linearmente Separáveis

Considere um conjunto de dados onde as classes não podem ser separadas por uma linha reta.

Classe + (círculo interno)
Classe - (região externa)

      • • • • •
    •           •
  •      +        •
  •               •
    •           •
      • • • • •

Nesse cenário, um modelo linear falha, pois não existe uma linha que separe corretamente as classes.

---

2º - Ideia do Kernel Trick

A solução do SVM é transformar os dados para um espaço de maior dimensão onde eles se tornam separáveis linearmente.

  • No espaço original → não separável
  • No espaço transformado → separável por um hiperplano

Essa transformação é feita por uma função \(\phi(x)\).

Problema: calcular \(\phi(x)\) explicitamente pode ser caro.

Solução: usar o Kernel Trick.

---

3º - O que é um Kernel?

Um kernel é uma função que calcula o produto interno no espaço transformado sem precisar calcular \(\phi(x)\):

\( K(x, x') = \phi(x) \cdot \phi(x') \)

Isso permite trabalhar em espaços de alta dimensão de forma eficiente.

---

4º - Função de Decisão

A função de decisão do SVM com kernel é:

\( f(x) = \sum_{i=1}^{n} \alpha_i y_i K(x_i, x) + b \)

Onde:

  • \(x_i\): pontos de treinamento
  • \(y_i\): rótulos (\(+1\) ou \(-1\))
  • \(\alpha_i\): coeficientes aprendidos
  • \(K(x_i, x)\): função kernel
  • \(b\): bias

A previsão é dada pelo sinal de \(f(x)\).

---

5º - Tipos de Kernel

Kernel Linear

\( K(x, x') = x \cdot x' \)

Equivalente ao SVM linear.

---

Kernel Polinomial

\( K(x, x') = (x \cdot x' + c)^d \)

  • \(d\): grau do polinômio
  • \(c\): constante
---

Kernel RBF (Gaussiano)

\( K(x, x') = \exp(-\gamma ||x - x'||^2) \)

  • \(\gamma\): controla a influência dos pontos

É o mais utilizado na prática.

---

6º - Intuição do Kernel RBF

O kernel RBF mede similaridade baseada na distância:

  • Pontos próximos → valor alto
  • Pontos distantes → valor próximo de 0

Isso cria fronteiras de decisão não lineares.

---

7º - Papel dos Support Vectors

Assim como no SVM linear, apenas alguns pontos influenciam a decisão:

  • Os pontos mais próximos da fronteira
  • Chamados de support vectors

Os demais pontos não afetam diretamente o modelo.

---

8º - Hiperparâmetros Importantes

  • C: controla a penalização de erros
    • C alto → menos erro, mais overfitting
    • C baixo → mais margem, mais generalização
  • \(\gamma\): controla a influência dos pontos
    • \(\gamma\) alto → fronteiras complexas
    • \(\gamma\) baixo → fronteiras suaves
---

9º - Interpretação Geométrica

No espaço original, a fronteira pode ser curva:

Classe + dentro do círculo
Classe - fora

Fronteira: curva fechada

No espaço transformado, essa fronteira se torna linear.

---

10º - Limitações

  • Escolha do kernel pode ser difícil
  • Escalabilidade limitada para grandes datasets
  • Interpretação difícil em kernels complexos
---

11º - Conclusão

O SVM com kernel permite lidar com dados não linearmente separáveis.

Usando o kernel trick, conseguimos trabalhar em espaços de alta dimensão sem custo computacional elevado.

É um modelo poderoso, especialmente com o kernel RBF, mas depende de uma boa escolha de hiperparâmetros.
