PCA (Principal Component Analysis)

1º - O Problema: Muitas Variáveis

Considere um conjunto de dados com duas variáveis:

Altura (cm) Peso (kg)
15050
16060
17065
18080
19090

Essas variáveis são correlacionadas.

O problema é:

  • Redundância de informação
  • Dificuldade de visualização em alta dimensão
  • Custo computacional maior

Queremos reduzir a dimensionalidade mantendo o máximo de informação possível.

---

2º - Ideia do PCA

O PCA cria novas variáveis chamadas componentes principais.

  • São combinações lineares das variáveis originais
  • São ortogonais (independentes entre si)
  • Capturam a maior variância possível
---

3º - Intuição Geométrica

Imagine os dados em um plano:

Peso (Y)
  ^
90|                •
80|            •
70|
60|        •
50|    •
   |
   +------------------------> Altura (X)
       150 160 170 180 190

Os pontos seguem uma direção principal.

O PCA encontra essa direção (linha inclinada) e projeta os dados nela.

---

4º - Centralização dos Dados

Primeiro, subtraímos a média de cada variável:

\( X_{centralizado} = X - \bar{X} \)

Isso move os dados para o centro (origem).

---

5º - Matriz de Covariância

Calculamos a matriz de covariância:

\( \Sigma = \frac{1}{n} X^T X \)

Ela mede como as variáveis variam juntas.

---

6º - Autovalores e Autovetores

O PCA resolve:

\( \Sigma v = \lambda v \)

  • \(v\): autovetores (direções)
  • \(\lambda\): autovalores (importância)

O maior autovalor indica a direção de maior variância.

---

7º - Componentes Principais

Ordenamos os autovalores:

  • 1º componente → maior variância
  • 2º componente → segunda maior (ortogonal ao primeiro)

Selecionamos os primeiros \(k\) componentes.

---

8º - Projeção dos Dados

Projetamos os dados nos novos eixos:

\( Z = XW \)

Onde:

  • \(X\): dados originais
  • \(W\): matriz com os autovetores
  • \(Z\): dados transformados
---

9º - Redução de Dimensionalidade

Se temos 2 variáveis, podemos reduzir para 1:

  • Usamos apenas o primeiro componente
  • Perdemos pouca informação

Isso simplifica o problema.

---

10º - Variância Explicada

A fração da variância explicada é:

\( \frac{\lambda_i}{\sum_j \lambda_j} \)

Usamos isso para decidir quantos componentes manter.

---

11º - Quando Usar PCA

  • Redução de dimensionalidade
  • Visualização de dados
  • Remoção de ruído
  • Pré-processamento para outros modelos
---

12º - Limitações

  • Assume relações lineares
  • Difícil interpretação das componentes
  • Sensível à escala dos dados
---

13º - Conclusão

O PCA transforma dados correlacionados em um novo conjunto de variáveis independentes.

Ele reduz dimensionalidade mantendo a maior quantidade possível de informação.

É uma das técnicas mais importantes em aprendizado não supervisionado.
