Risco Empírico

1º - Objetivo: Medir o Erro do Modelo

Em Machine Learning, queremos treinar modelos que façam boas previsões.

Mas surge uma pergunta fundamental:

Como saber se um modelo está indo bem?

Para isso, precisamos medir o erro entre o que o modelo prevê e o valor real.

2º - Ideia Intuitiva

Suponha que temos os seguintes dados:

Valor Real (y) Previsão (\(\hat{y}\))
32.5
55.5
22
76

Podemos calcular o erro de cada previsão.

3º - Função de Perda (Loss Function)

Uma forma comum de medir erro é o Erro Quadrático:

\( L(y, \hat{y}) = (y - \hat{y})^2 \)

Calculando para cada ponto:

  • (3 - 2.5)² = 0.25
  • (5 - 5.5)² = 0.25
  • (2 - 2)² = 0
  • (7 - 6)² = 1

4º - Definição de Risco Empírico

O Risco Empírico é simplesmente a média dos erros no conjunto de dados:

\( R_{emp} = \frac{1}{n} \sum_{i=1}^{n} L(y_i, \hat{y}_i) \)

No nosso exemplo:

\( R_{emp} = \frac{0.25 + 0.25 + 0 + 1}{4} = 0.375 \)

5º - Interpretação

  • O risco empírico mede o desempenho do modelo nos dados observados.
  • Quanto menor, melhor o modelo se ajusta aos dados.
  • Ele depende diretamente da função de perda escolhida.

6º - O Objetivo do Aprendizado

Treinar um modelo é, essencialmente, minimizar o risco empírico:

\( \min_{\theta} R_{emp}(\theta) \)

Onde \(\theta\) representa os parâmetros do modelo.

7º - Limitação Importante

Minimizar o risco empírico não garante bom desempenho em novos dados.

Isso pode levar ao problema de overfitting, onde o modelo "decora" os dados.

8º - Conexão com Generalização

O ideal seria minimizar o Risco Verdadeiro, que considera todos os dados possíveis:

\( R_{true} = \mathbb{E}[L(y, \hat{y})] \)

Como não temos acesso a todos os dados do mundo, usamos o risco empírico como aproximação.

9º - Conceitos Importantes

  • Loss Function: mede erro individual
  • Risco Empírico: média dos erros no dataset
  • Minimização: objetivo do treinamento
  • Overfitting: bom no treino, ruim na prática

10º - Conclusão

O risco empírico é a base matemática do aprendizado em Machine Learning.
Ele conecta dados, erro e otimização em um único conceito,
servindo como ponto de partida para praticamente todos os algoritmos.
