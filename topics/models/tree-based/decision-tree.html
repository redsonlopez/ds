Árvore de Decisão

1º - O Problema: Aprovar ou Reprovar

Considere o seguinte conjunto de dados de 5 alunos:

Horas de Estudo Faltas Aprovado?
25Não
32Sim
16Não
41Sim
24Não

Nosso objetivo é prever se um aluno será aprovado com base nas características.

2º - Ideia da Árvore de Decisão

A árvore funciona como um fluxograma de perguntas:

  • Nó: A pergunta (ex: Horas de estudo ≥ 3?).
  • Ramo: A resposta (Sim ou Não).
  • Folha: A previsão final (Aprovado ou Reprovar).

3º - Como Escolher a Melhor Pergunta?

Usamos a Impureza de Gini.
Ela mede o quão "misturados" estão os dados em um grupo.
Se o Gini é 0, o grupo é puro (todos são da mesma classe).

\( Gini = 1 - \sum p_i^2 \)

Onde \(p_i\) é a proporção de cada classe.

4º - Cálculo da Impureza Inicial

Antes de qualquer divisão, temos 3 "Não" e 2 "Sim":

\( p_{Não} = \frac{3}{5}, \quad p_{Sim} = \frac{2}{5} \)

\( Gini = 1 - \left(\left(\frac{3}{5}\right)^2 + \left(\frac{2}{5}\right)^2\right) \)

\( Gini = 1 - (0.36 + 0.16) = 0.48 \)

5º - Testando um Corte: Horas ≥ 3

Grupo 1: Horas ≥ 3 (Sim)

Contém 2 alunos, ambos "Sim". Logo, é um grupo puro: Gini = 0.

Grupo 2: Horas < 3 (Não)

Contém 3 alunos, todos "Não". Logo, também é puro: Gini = 0.

6º - Impureza Após o Corte

Para saber se o corte foi bom, calculamos a impureza final ponderada pelo tamanho de cada grupo:

\( Gini_{final} = \left(\frac{2}{5} \times 0\right) + \left(\frac{3}{5} \times 0\right) = \mathbf{0} \)

Como a impureza caiu de 0.48 para 0, tivemos um Ganho de Informação máximo!

7º - Construção da Árvore

A representação visual da lógica aprendida pelo modelo:

          [Horas ≥ 3?]
            /      \
        Sim          Não
       /              \
 [Aprovado]       [Reprovado]

Se o problema fosse mais complexo, a árvore continuaria fazendo perguntas nos ramos impuros:

          [Horas ≥ 3?]
            /      \
        Sim          Não
       /              \
 [Aprovado]       [Faltas ≤ 2?]
                    /      \
                Sim          Não
               /              \
         [Aprovado]       [Reprovado]

8º - Interpretação

  • A árvore aprendeu uma regra simples baseada nos dados.
  • Ela divide o problema em decisões menores.
  • Não precisamos de fórmula explícita como na regressão.

9º - Conceitos Importantes

  • Nó: pergunta (ex: Horas ≥ 3)
  • Ramo: resultado da pergunta
  • Folha: decisão final (Sim ou Não)
  • Impureza: mede o quão "misturado" está o conjunto

10º - Conclusão

O modelo busca automaticamente os cortes que mais reduzem a impureza.
Essa simplicidade torna a Árvore de Decisão um dos modelos mais explicáveis em Machine Learning,
servindo de base para algoritmos poderosos como Random Forest.
